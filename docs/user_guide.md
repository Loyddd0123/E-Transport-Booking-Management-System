# User Guide

## Passenger Web System
1. Open the web page.
2. Login using the sample passenger account.
3. View schedules.
4. Click Book and enter the number of seats.
5. View the booking status in the recent bookings table.

## Admin Windows System
1. Open the C# project in Visual Studio.
2. Run the application.
3. Select a module from the dropdown.
4. Click Refresh to load API data.
5. Open bookings and select a booking row.
6. Click Mark Booking Paid or Cancel Booking.
7. Refresh the web app to see the synchronized change.
